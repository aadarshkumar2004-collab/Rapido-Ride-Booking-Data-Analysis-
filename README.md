# Rapido Ride Booking Data Analysis

## Project Overview
This project analyzes ride booking data to understand customer behavior, booking trends, ride performance, and cancellation patterns.  
The dataset contains more than **100,000 ride booking records**, which were analyzed using **SQL, Microsoft Excel, and Power BI** to generate business insights and visual dashboards.

The objective of this project is to simulate a **real-world ride-hailing business analysis** and identify key operational and customer insights.

---

## Problem Statement
Ride-hailing platforms generate large volumes of operational data.  
The goal of this project is to analyze ride booking data and answer key business questions such as:

- How many rides were successfully completed?
- What are the most common reasons for ride cancellations?
- Which vehicle types are used most frequently?
- Who are the top customers based on booking value?
- What is the average ride distance by vehicle type?
- How do customer and driver ratings compare?
